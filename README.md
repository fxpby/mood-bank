# 情感储蓄罐

移动端优先的 PWA 情绪观察与自我稳定工具。当前 P0 聚焦单人、本地优先的核心闭环：设置一个情感空间、在被触发时先回到自己、记录事实与解释、保存回到自己的练习，并从记录中派生 Connection / Self / Energy 三个储蓄罐摘要。

## 当前边界

- 本项目是前端本地应用，没有后端、登录、云同步、遥测、推送或 AI 分析。
- 数据保存在当前浏览器的 `localStorage` 中。更换设备、清理浏览器数据、隐私模式或浏览器权限变化都可能导致记录丢失。
- 当前提供 manifest 和移动端 PWA 外壳，但没有注册 service worker。因此它是本地优先，不是完整离线优先；首次加载和资源缓存仍依赖浏览器默认行为。
- 储蓄罐摘要由记录和练习中的影响事件派生，不作为权威分数持久化。

## 本地开发

```bash
npm install
npm run dev
```

默认开发地址是 `http://127.0.0.1:5173`。如果该端口被占用，Vite 会使用下一个可用端口。

## 验收命令

```bash
npm run typecheck
npm test
npm run build
```
